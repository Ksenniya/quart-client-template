### User Requirement Document: Data Analysis and Reporting for London Houses Data

#### User Stories
1. **As a user**, I want to download the London Houses Data so that I can analyze it.
2. **As a user**, I want to analyze the data using Pandas so that I can generate insights.
3. **As a user**, I want to save a report based on the analysis so that I can review and share it later.

```mermaid

    A[Start] --> B[Download London Houses Data]
    B --> C[Analyze Data with Pandas]
    C --> D[Generate Reports]
    D --> E[Save Reports]
    E --> F[End]
```

### API Outline

#### London Houses Data API
- **POST /london-houses-data**
  - **Description**: To upload the London Houses Data.
  - **Action**: `entity_service.add(current_entity)`

- **GET /london-houses-data**
  - **Description**: To fetch the existing London Houses Data.
  - **Action**: `entity_service.get(current_entity)`

---

#### Analysis Report API
- **POST /analysis-report**
  - **Description**: To upload the analysis report.
  - **Action**: `entity_service.add(current_entity)`

- **GET /analysis-report**
  - **Description**: To fetch the generated analysis report.
  - **Action**: `entity_service.get(current_entity)`
