# ### User APIs for Cyoda Deployment and Environment Configuration
# 
# #### 1. API for Creating a Cyoda Environment
# 
# - **POST /deploy/cyoda-env**
#     - **Request Body:**
#       ```json
#       {
#         "user_name": "test"
#       }
#       ```
#     - **Description:** Creates a new Cyoda environment for the specific user.
#     - **Service Call:** 
#       ```python
      entity_service.add(current_entity)  # where current_entity is a new Cyoda environment entity
#       ```
# 
# #### 2. API for Deploying User Application
# 
# - **POST /deploy/user_app**
#     - **Request Body:**
#       ```json
#       {
#         "repository_url": "http://....",
#         "is_public": "true"
#       }
#       ```
#     - **Description:** Initiates the deployment of the user application by adding it to the build queue.
#     - **Service Call:** 
#       ```python
      entity_service.add(current_entity)  # where current_entity is a new user application entity
#       ```
# 
# #### 3. API for Retrieving Cyoda Environment Status
# 
# - **GET /deploy/cyoda-env/status/{id}**
#     - **Description:** Retrieves the status of the specified Cyoda environment.
#     - **Service Call:** 
#       ```python
      entity_service.get(current_entity)  # where current_entity represents the specified cyoda environment by id
#       ```
# 
# #### 4. API for Retrieving Cyoda Environment Statistics
# 
# - **GET /deploy/cyoda-env/statistics/{id}**
#     - **Description:** Fetches statistics for the specified Cyoda environment.
#     - **Service Call:** 
#       ```python
      entity_service.get(current_entity)  # where current_entity represents the specified cyoda environment by id
#       ```
# 
# #### 5. API for Retrieving User Application Status
#

#     - **Service Call:**

      entity_service.get(current_entity)  # where current_entity represents the specified user application by id

      entity_service.get(current_entity)  # where current_entity represents the specified user application by id
