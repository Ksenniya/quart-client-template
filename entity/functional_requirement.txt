### Functional Requirements

#### User Stories

1. **User Authentication**
   - **As a** user,
   - **I want** to authenticate using a Bearer token,
   - **So that** I can access deployment features securely.

   **API Endpoint:**
   - **POST** `/deploy/cyoda-env`
     - **Request:**
       ```json
       {
         "user_name": "test"
       }
       ```
     - **Response:** (Assuming successful authentication)
       ```json
       {
         "token": "your_auth_token_here"
       }
       ```

2. **Create Deployment for Env Configuration**
   - **As a** user,
   - **I want** to create an environment with user-defined configurations,
   - **So that** I can manage my deployments.

   **API Endpoint:**
   - **POST** `/deploy/cyoda-env`
     - **Request:**
       ```json
       {
         "user_name": "test"
       }
       ```
     - **Response:**
       ```json
       {
         "build_id": "id_of_the_created_build"
       }
       ```

3. **Deploy User Application**
   - **As a** user,
   - **I want** to deploy a user application with specific repository details,
   - **So that** I can manage my applications.

   **API Endpoint:**
   - **POST** `/deploy/user_app`
     - **Request:**
       ```json
       {
         "repository_url": "http://...",
         "is_public": "true"
       }
       ```
     - **Response:**
       ```json
       {
         "build_id": "id_of_the_created_user_app_build"
       }
       ```

4. **Get Status of Deployment**
   - **As a** user,
   - **I want** to check the status of an existing deployment,
   - **So that** I can know the current state of my application.

   **API Endpoint:**
   - **GET** `/deploy/cyoda-env/status/{id}`
     - **Response:**
       ```json
       {
         "status": "Building",
         "repository_url": "http://...",
         "is_public": "true"
       }
       ```

5. **Get Statistics of a Deployment**
   - **As a** user,
   - **I want** to retrieve deployment statistics,
   - **So that** I can analyze the performance of my application.

   **API Endpoint:**
   - **GET** `/deploy/cyoda-env/statistics/{id}`
     - **Response:**
       ```json
       {
         "repository_url": "http://...",
         "is_public": "true",
         "statistics": {
           "buildDuration": "5 minutes",
           "successRate": "95%"
         }
       }
       ```

6. **Cancel User Application Deployment**
   - **As a** user,
   - **I want** to cancel an ongoing deployment,
   - **So that** I can manage my build queue effectively.

   **API Endpoint:**
   - **POST** `/deploy/cancel/user_app/{id}`
     - **Request:**
       ```json
       {
         "comment": "Canceling a queued build",
         "readdIntoQueue": false
       }
       ```
     - **Response:**
       ```json
       {
         "message": "Build cancelled successfully."
       }
       ```
